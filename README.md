# Brand Logo Detection
Учебный проект по компьютерному зрению: детектирование логотипов брендов на изображениях с использованием датасета OpenLogo.

## Цель проекта
Реализировать полный pipeline для задачм object detection:
- подготовка и анализ датасета;
- предобработка изображений и аннотаций;
- обучение и сравнение нескольких моделей детектирования объектов;
- расчет метрик качества;
- анализ ошибок моделей;
- подготовка отчета и презентации;

## Датасет
Планируется использовать OpenLogo dataset. 
Для учебной практики будет выбран supervised subset из 20-32 классов логотипов.

Raw dataset не хранится в репозитории.

## Модели
Планируется сравнить минимум 5 моделей:
- YOLOv8n
- YOLOv8s
- YOLOv5s
- Faster R-CNN ResNet50-FPN
- SSD300 или RT-DETR

## Метрики
- mAP@50
- Precision
- Recall
- F1-score
